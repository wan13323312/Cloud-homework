## 前端用法介绍
## 搜索功能
1. 在搜索框里输入要查询的概念（如"熵”），点击“确认”按钮，开始搜索。
2. 调用后端generate api
3. 在等待大约3-5分钟后，我们可以在图像区域看到一张以查询概念为中心的知识图谱。
4. 边的颜色越红，关联强度越高

## 高亮功能
1. 高亮节点：鼠标悬浮在节点上，与该节点相邻的节点与边都会被高亮。
2. 高亮边：鼠标悬浮在边上，该边及其两个节点都会被高亮。同时在边信息栏会显示边的来源，目标，关系和强度

## 双击扩展：
1. 双击节点，调用后端query_db的api
2. 扩展出与该节点相关的更多节点

## 寻路模式
1. 点击“开启寻路模式”，边的颜色全部变为绿色，图的透明度提升。
2. 在选择框里选择两个节点。
3. 点击“开始寻路”按钮。
4. 在图中生成一条红色高亮路径，生成方式：路径的最小强度值最大
